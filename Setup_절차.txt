
아래의 프로그램들을 순서대로 관리자 권한으로 실행 시키시면 됩니다.

1.  "Setup__MODULE_LINKER\setup.exe"
  * 해당 프로그램은 C Drive에 설치됩니다.
2.  "Setup__SATELLITE_5P\setup.exe"
  * 해당 프로그램은 D Drive에 설치됩니다.
3.  "Update__LICENCE_5P (최신날짜)\Update__M_Linker.exe"
  # 해당 실행하면 수행되는 내용
    1) 최신 등록된 Licence Key 들이 등록됩니다.
	  주의) Licence Key는 Wi-Fi 옵션을 켜야 인식 됩니다.
    2) RPM 측정 프로그램이 실행되는데 필요한 Local Path가 등록됩니다.
    3) RPM 측정 프로그램에 추가된 기능들이 자동 업데이트 됩니다.


바탕화면에 있는 아래의 Icon를 실행
  SUI__LKG3000_5P.exe
  EQP__LKG3000_5P.exe


* 화면 해상도가 1680 x 1050 인 경우에는 아래의 Setup 프로그램을 설치하세요.
  "Setup__SUI_LKG3000_5P-1680_1050\setup.exe"
  바탕화면의 Icon은 "SUI_P5-1680_1050" 입니다.


주의) LK-G3000 Driver는 센서엡체에서 받은 Driver로 설치하세요.
  "LK-H1W Navigator LK-G3.zip" 은 이전 버전일 수 있습니다.
